# Quick Task 260528-u67: Fix DPI Scaling — Fit-Page, Scroll, and Geometry Sizes

**One-liner:** DPI-aware fit-page zoom formula (`canvasPx / (heightPt * 96/72 * dpiScale)`), matching scroll clamp, ZoomIn width cap, and per-frame geometry size scaling via `_currentDpiScaleF` field.

## What Was Done

### Task 1 — MainViewModel DPI math and PdfCanvasViewModel property exposure

`PdfCanvasViewModel` gained two public read-only properties:
- `DpiScale` — exposes `_dpiScale` so MainViewModel can include it in zoom formulas
- `CanvasWidthPx` — exposes `_canvasWidthPx` so ZoomIn can cap at the page-width zoom

`MainViewModel` changes:
- `ApplyFitPage`: zoom formula changed from `canvasPx / (heightPt * 96/72)` to `canvasPx / (heightPt * 96/72 * dpiScale)`. On a 2x screen the old formula produced half the correct zoom.
- `ClampScrollOffset`: `pageHeightPx` now includes `dpiScale`, so the max-scroll ceiling matches the physical canvas height.
- `ScrollThumbTopRatio`: same fix as ClampScrollOffset — `pageHeightPx` includes `dpiScale`.
- `ZoomIn`: after finding the next step, caps at `canvasW / (widthPt * dpiScale * 96/72)` — the zoom at which the page width exactly fills the canvas. Guards `canvasW > 0 && widthPt > 0` prevent division by zero per T-u67-02.

### Task 2 — Geometry and ghost preview DPI scaling

`GeometryLayerViewModel`:
- Added `_lastDpiScale = 0.0` (forces first-run update) and `_currentDpiScaleF = 1.0f` (frame-local scale).
- `Draw()` signature changed to `Draw(SKCanvas, CoordinateMapper?, double dpiScale = 1.0)`.
- At the top of `Draw()`, when `dpiScale` differs from `_lastDpiScale` by more than 0.001, all cached paint stroke widths and font sizes are updated in-place (no allocations — SKFont.Size is settable in SkiaSharp 3.x).
- All hardcoded physical radii and lengths multiplied by `_currentDpiScaleF`: PointObject ring (8f) and dot (4f), CircleObject center dot (4f), DrawSubPointDot filled dot (8f) and active ring (14f), DrawTextLabel selection padding (4f), DrawProtractor tick lengths (24f/13f/5f), label radii offsets (32f/58f), label clamp floor (8f), crosshair circle (4f) and line extents (8f/3f).

`PdfCanvasViewModel`:
- `Paint()` call site updated to `_geometryLayer.Draw(canvas, _coordinateMapper, _dpiScale)`.
- `DrawGhostPreview`: `float ds = (float)_dpiScale` captured once; all StrokeWidths (2f), anchor dot radius (5f), anchor ring radius (12f), snap ring StrokeWidth (2f) and radius (18f), snap dot radius (5f) multiplied by `ds`.
- `DrawGhostProtractor`: `float dps = (float)_dpiScale` captured; arc paint StrokeWidth (2f), arm line StrokeWidth (1.5f), center dot radius (4f) all multiplied by `dps`.
