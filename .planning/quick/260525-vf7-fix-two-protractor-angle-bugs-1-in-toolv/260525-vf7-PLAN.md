Key conventions established in Phase 03:
- BaselineAngleDeg is screen-space CW so SkiaSharp RotateDegrees applies without Y-flip.
- Arc occupies the negative-Y half of local canvas space (DrawProtractor body paint at y=-3 to -8).

- SKPaint cache: all paints are readonly fields, never allocated per frame.

// When IsFlipped, the outer scale's 0° is at the left end of the baseline (-180° in local space).
// Draw arc starting from -180°, sweeping CW (positive in Skia) by measuredAngleDeg.

float arcStartDeg, arcSweepDeg, midAngleRad;
if (isFlipped)
{
    arcStartDeg  = -180f;
