# 4. Core Object Model (CRITICAL)

Everything must be treated as:

- Points  
- Lines / segments  
- Circles / arcs  
- Shapes (polygons)  
- Measurements (angle, length, etc.)  
- Labels / annotations  

---

# 5. Geometry Construction Tools

## 5.1 Basic Construction

| Tool | Interaction | Notes |
|-----|------------|------|
| Point | Click location | Snaps to intersections |
| Line | Click A → click B | Snap options |
| Segment | Same as line | Finite |
| Ray | Same as line | One direction |
| Circle | Center → radius point | Compass equivalent |
| Arc | 3-point selection | Optional |

---

## 5.2 Advanced Construction

| Tool | Function | Why Important |
|------|--------|--------------|
| Midpoint Tool | Create midpoint of segment | Used in constructions |
| Perpendicular Line | Line at 90° | Common exam task |
| Parallel Line | Maintain slope | Graph/geometry |
| Angle Bisector | Split angle evenly | High exam relevance |
| Locus Tool | Trace movement paths | Higher-tier maths |
| Auxiliary Line Tool | Add helper lines | Critical for problem solving |

# 7. Protractor (High Priority)

## Features:
- Snap to vertex
- Align to line automatically
- Rotate via buttons (±1°, ±5°)
- Flip inner/outer scale
- Lock position

## Visual:
- High contrast
- Large readable markings
- Transparent overlay

## Constraints:
- No angle value in exam mode  
- Magnifier allowed

# 8. Transformations (CRITICAL)
